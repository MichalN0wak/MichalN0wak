## Hi, I'm Michał 👋

an **IT Project Manager** who combines PM, BA, and QA through discovery → delivery → validation.

- **Discovery & requirements:** workshops and discovery; Product Requirements Documents (PRDs), product phasing (MVP vs later), dependencies; backlog and priorities, user stories, use cases and acceptance criteria that are clear enough to estimate, implement, and verify.

- **Validation:** manual testing, API testing, sign-off against the agreed specification.
